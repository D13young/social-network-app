# SocialNetworkApp

Социальная сеть с поддержкой различных ролей пользователей, постов и сообществ. Приложение построено на ASP.NET Core и позволяет пользователям создавать контент, взаимодействовать в сообществах, а также имеет расширенную систему аутентификации и авторизации.

---

## Технологии

- .NET 8.0 (ASP.NET Core)
- Entity Framework Core 9 с SQLite (для основных данных)
- ASP.NET Core Identity для поддержки пользователей и ролей
- Локализация: русский и английский языки
- Email-сервис для отправки писем (использует библиотеку Resend)

---

## Установка и запуск

### Клонирование репозитория

```bash
git clone https://github.com/D13young/social-network-app.git
cd SocialNetworkApp
```

### Настройка конфигурации

Скопируйте файл `appsettings.json.example` в `appsettings.json` и укажите ваши значения:

- `"ConnectionStrings:ApplicationDbContextConnection"` — строка подключения к БД SQLite по умолчанию `Data Source=SocialNetwork.db`
- `"Resend:ApiKey"` — API ключ для сервиса отправки email (получите на https://resend.com)
- `"Resend:FromEmail"` — адрес электронной почты, с которого отправляются письма

### Применение миграций базы данных

Приложение автоматически применит миграции при запуске, но можно выполнить вручную:

```bash
dotnet ef database update
```

### Запуск приложения

```bash
dotnet run
```

Приложение будет доступно по адресу http://localhost:5000 (или другому, указанному в конфигурации).

## Основные возможности

- Регистрация и подтверждение пользователей с ролями:
  - Пользователь (User)
  - Модератор (Moderator)
  - Администратор (Admin)
- Управление постами, комментариями и сообществами
- Ролевая авторизация с политиками доступа
- Локализация интерфейса на русский и английский языки
- Отправка email, например для подтверждения регистрации

---

## Контакт и настройки email

Для работы отправки почты укажите в `appsettings.json` параметры:

```json
"Resend": {
  "ApiKey": "ВАШ_API_КЛЮЧ",
  "FromEmail": "your-email@example.com"
}
```

Получить API ключ можно на сервисе [Resend](https://resend.com).

## Структура проекта

- **Areas/Identity** — страницы и логика для аутентификации и управления пользователями
- **Controllers/** — контроллеры MVC для основных сущностей (посты, комментарии, сообщества)
- **Data/** — контекст базы данных и миграции
- **Models/** — модели данных
- **Views/** — представления Razor для UI
- **wwwroot/** — статические файлы (css, js, изображения)
- **Services/** — дополнительные сервисы (например, отправка email)
